Kivy

Kivy 是可以构建跨平台（IOS，MACOS，Android，Windows）桌面应用的python框架。

具体介绍请看官方文档 https://kivy.org/

安装Kivy

  1. 首先在创建一个python的虚拟环境。
    python3 -m venv env

  2. 用pip安装
    pip install -i https://pypi.doubanio.com/simple/ --trusted-host pypi.doubanio.com kivy
